# Frontend Mentor Projects

A collection of responsive web components and landing pages built as part of my Frontend Mentor challenge journey. These projects demonstrate my HTML and CSS skills, focusing on creating pixel-perfect, responsive designs using modern layout techniques.

## About This Repository

This repository contains my solutions to various Frontend Mentor challenges. Each project emphasizes clean code, responsive design principles, and attention to detail in replicating professional UI designs.

## Technologies Used

- HTML
- CSS
- Bootstrap
- Responsive Design Principles

## Key Learning Outcomes

Through these challenges, I've developed and refined skills in:

- Building responsive layouts that work seamlessly across different screen sizes
- Implementing modern CSS layout systems (Flexbox and Grid)
- Writing semantic HTML markup
- Creating pixel-perfect designs from provided mockups
- Understanding component-based design patterns
- Optimizing CSS for maintainability and performance

## Project Structure

Each project is contained in its own directory with:

- Source HTML and CSS files
- Assets (images, icons, fonts)

## Getting Started

To view any project locally:

1. Clone this repository
```bash
git clone https://github.com/Yosef-Ayman/Frontend-Mentor-Challenges.git
```

2. Navigate to the specific project directory
```bash
cd Frontend-Mentor-Challenges/project-name
```

3. Open the `index.html` file in your browser
